The Query Search allows you to find the closest semantic matches for a given variable within a selected terminology. By leveraging embeddings from LLMs and PLMs, Kitsune helps you identify relevant ontology or standard concepts efficiently.

To perform a search, enter a query (e.g., Diabetes), select a language model and a terminology from the dropdown lists, and submit your request. The system will retrieve the closest matches from the chosen terminology, ranked by similarity. You can explore the results and adjust the number of items displayed per page using the table controls below.
